# 🐧 Crikitiso

> Distribuciones Linux personalizadas para cada necesidad.

**Crikitiso** es una empresa ficticia creada como Proyecto Intermodular de 2º de Grado Medio de Sistemas Microinformáticos y Redes (SMR). Tomamos distribuciones Linux base como Pop!_OS y Ubuntu y las modificamos con **Cubic** para adaptarlas a tres perfiles concretos: gaming, edición creativa y desarrollo de software.
